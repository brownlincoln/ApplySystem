作为主攻Java的猿，最近学了一点儿vue2.0，浑身难受，想做点儿什么。正好在秋招，需要整理网申记录（难受，投了一大堆，没有一个offer），于是，做了这个简洁的网申记录网页。前端学得不好，做的不好。如图。

![主页图片](https://github.com/brownlincoln/ApplySystem/blob/master/home_show.PNG?raw=true)

## 技术
这里是后端系统。
SpringBoot1.5.6 + MySQL5.7 + MyBatis3.x + (tomcat)

前端技术见另一个项目[apply_sys](https://github.com/brownlincoln/apply_sys)： Vue2.0 + axios

## 补充
项目并不完善，但可以使用，管理网申记录很方便啊.

使用建议：  
1）使用中，尽量不要点“退出”。点了退出，下次还要输入帐号和密码。不点的话，下次可以直接进入。  
2）如果使用中，有问题了，可以在issue中找我，或者发邮件chrisliu1314@qq.com
